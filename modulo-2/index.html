Exercicio módulo 2

O propósito desse exercicio é treinar criação de arquivos e subida para o github

Tópicos abordados nesse modulo

  • Criação de repositório no Github
  • Git clone - baixar seu repositório na sua maquina
  • Git add - Adicionar modificações no seu repositório local
  • Git commit -m - Adicinar uma mensagem "commit"
  • Git push - subir a modificação que está no seu repositório local para o repositório remoto
